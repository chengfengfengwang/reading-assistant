# reading-assistant
一个谷歌浏览器插件。使用OpenAI Agent和当前网页对话。

## 下载使用
1. 从releases下载
2. 解压下载文件，在浏览器扩展加载该文件
3. 输入自己的OpenAI Key

## development
1. git clone 
2. npm install
3. npm run dev
4. open chrome and load exension from dist folder
